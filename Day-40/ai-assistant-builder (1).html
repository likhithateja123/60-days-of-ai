📘 How it was built ▾

System Prompt

Assistant type: Coding assistant for beginners.
Input: Free-form — code snippets, plain text descriptions, or error/stack traces.
Output: Step-by-step guidance without giving full working code, so learners build understanding rather than copy-paste answers.
Tone: Friendly & encouraging.

Mode: Fully offline demo. This build simulates Claude's responses using predefined sample data matched to your question's keywords, so it works entirely without an API key or internet connection.

Built as a single HTML file with vanilla JS. A small keyword-matching function picks the most relevant canned response (or a friendly fallback if nothing matches) and displays it after a short simulated delay, so the loading state feels realistic. Markdown-style responses are lightly parsed client-side for headings, code blocks, lists, and bold text. Swap the demo function for a live fetch() call to the Claude API whenever you're ready to go live.
