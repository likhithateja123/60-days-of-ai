# Day 20 – Face Puzzle Game with Claude AI

## Objective

Build a fully functional Face Puzzle Game using Claude AI and front-end web technologies. The application captures a user's photo through the webcam and converts it into an interactive puzzle game.

## Tools Used

* Claude AI
* HTML
* CSS
* JavaScript
* GitHub

## Features Implemented

* Webcam access using getUserMedia()
* Live camera preview
* Capture photo functionality
* Difficulty levels (3×3, 4×4, 5×5)
* Puzzle piece generation from captured image
* Random puzzle scrambling
* Drag-and-drop support
* Touch gesture support for mobile devices
* Timer and move counter
* Correct pieces tracker
* Automatic win detection
* Results screen with performance summary
* LocalStorage leaderboard
* Responsive user interface
* Retake Photo, Play Again, and New Photo options

## Learning Outcomes

* Learned how browser camera APIs work.
* Understood image processing using HTML Canvas.
* Explored puzzle generation and piece randomization techniques.
* Implemented drag-and-drop and touch interactions.
* Used localStorage to save leaderboard data.
* Improved understanding of responsive front-end development.
* Experienced how AI can accelerate application development and prototyping.

## Challenges Faced

* Managing webcam permissions across browsers.
* Ensuring drag-and-drop worked smoothly on both desktop and mobile devices.
* Maintaining accurate timer and move tracking.
* Creating a responsive layout for different screen sizes.

## Conclusion

This project demonstrated how AI-assisted development can be used to rapidly build interactive web applications. By combining webcam functionality, image processing, and puzzle mechanics, I gained practical experience in front-end development and learned how AI can support the software development process.
