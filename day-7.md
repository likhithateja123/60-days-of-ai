# Day 07 - Claude Usage Strategy

## Objective

Learn how to use different Claude models and effort levels effectively based on personal goals and daily tasks.

## My Profile

* Student
* Daily Claude User
* Learning, Coding, Career Preparation, Research, AI Projects, ServiceNow Learning

## Recommended Primary Model

Claude Sonnet 4.6

### Why This Model Fits Me

* Balanced intelligence and speed
* Suitable for learning and coding
* Good for research and career preparation
* Best overall model for daily student usage

## When to Use Each Model

### Haiku

* Quick questions
* Simple definitions
* Fast responses

### Sonnet

* Learning concepts
* Coding projects
* Debugging
* Documentation
* Daily usage

### Opus

* Complex projects
* System design
* Advanced reasoning
* Deep analysis

## Recommended Effort Levels

### Low

Quick answers and simple tasks

### Standard

Daily learning and coding

### High

Debugging and project planning

### Max

Complex research and advanced problem solving

## Key Learnings

* Different Claude models serve different purposes.
* Sonnet is the best balance for most student tasks.
* Higher effort levels should be used only when needed.
* Choosing the right model improves productivity.

## Final Recommendation

For most of my work, Claude Sonnet 4.6 with Standard effort provides the best balance of quality, speed, and efficiency.
